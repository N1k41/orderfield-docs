Pages

Default pages

The following default pages can be configured in OrderField:

  • Home (home)
  • Catalog page (catalog)
    • Item tile item_catalog
    • Item detail page item_details
  • Favorites favorites
  • Orderhistory orderhistory
  • Checkout checkout
  • Search search

Home page

Basic configuration

{
    "code" : "home",
    "i18n" : [],
    "name" : "Home",
    "sortOrder" : 1,
    "type" : "home"
}

Available options

  • Banner (one)
  • Blocks (multiple)

Catalog page

Basic configuration

{
    "code" : "catalog",
    "i18n" : [],
    "name" : "Catalog",
    "sortOrder" : 1,
    "type" : "catalog"
}

Available options

  • Banner (one)
  • Facetgroups
