Pages

The following default pages can be configured in OrderField:

  • Home
  • Catalog
    • Item tile
    • Item detail
  • Favorites
  • Orderhistory
  • Checkout
  • Search

Home

Basic configuration

{
    "code" : "home",
    "i18n" : [],
    "name" : "Home",
    "sortOrder" : 1,
    "type" : "home"
}

Attributes

  • banner
  • blocks

Catalog

Basic configuration

{
    "code" : "catalog",
    "i18n" : [],
    "name" : "Catalog",
    "sortOrder" : 1,
    "type" : "catalog"
}

Attributes

  • showThumbnails
  • facetgroups
  • banner

Item tile

Basic configuration

    "sortOrder" : 1,
    "type" : "catalog"
}
