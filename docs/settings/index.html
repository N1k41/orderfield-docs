Settings

The following settings can be applied.

Tenant settings

Collection: tenant > tenant__settings

SettingRemarks
ga_tracking_id
GA Tracking ID

Example
{"key" : "ga_tracking_id", "value" : "UA-87774626-1"}
gtm_tracking_id
GTM Tracking ID

Example
{"key" : "gtm_tracking_id", "value" : "UA-87774626-1"}
quantity_picker
Indicates which quantity picker should be used

Example
{"key" : "quantity_picker", "value" : "relative"}		Values
absolute ; relative
MAIL_ORDER_CONFIRMATION
Settings of the order confirmation emails

Sub-attributes
  • mail_from
    From-address
  • mail_cc (array)
    CC to
  • mail_bcc (array)
    BCC to
  • default_template_id
    Postmark template ID
  • template_ids (array)
    Postmark template ID per language
  • logo_url
    Log URL
  • mail_sender_name
    Name of the sender

Page settings

SettingLocations
showThumbnails
Indicates that thumbnails should be shown

Example
"platform" : "all", "key" : "showThumbnails", "value" : true		tenant__pages > catalog
showFavoriteIcon
Indicates that favorite icons should be shown

Example
"platform" : "all", "key" : "showFavoriteIcon", "value" : true		tenant__pages > item_catalog
tenant__pages > item_details
showAttributes
Determines which attributes should be shown

Example
"platform" : "all", "key" : "showAttributes", "value" : [ ]		tenant__pages > item_catalog
tenant__pages > item_details
showFacets
Determines which facets should be shown

Example
"platform" : "all", "key" : "showAttributes", "value" : ["scent", "color"]		tenant__pages > item_catalog
tenant__pages > item_details
showSalesOrderAddressInfo
Determines if address info for previous orders is available

Example
"platform" : "all", "key" : "showSalesOrderAddressInfo", "value" : true		tenant__pages > item_details
showDeliveryColumn
Determines if delivery column is shown

Example
"platform" : "all", "key" : "showDeliveryColumn", "value" : true		tenant__pages > checkout
showTermsAndConditionsCheckbox
Determines if terms checkbox is shown

Example
"platform" : "all", "key" : "showTermsAndConditionsCheckbox", "value" : true		tenant__pages > checkout
customerReferenceMaxLength
Determines max length of the reference field

Example
"platform" : "all", "key" : "customerReferenceMaxLength", "value" : 25		tenant__pages > checkout
showUomInfoForUoms
Determines to which customer group the pallet calculation is shown

Example
"platform" : "all", "key" : "showUomInfoForUoms ", "value" : ["PL"]		tenant__pages > checkout
showOrderType
Determines if Order Type choice is shown

Example
"platform" : "all", "key" : "showOrderType", "value" : false		tenant__pages > checkout
